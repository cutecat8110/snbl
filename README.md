## 🚀 安裝和使用

### 系統需求

建議 Node.js 16.20.2

### 取得專案

```sh
git clone https://github.com/cutecat8110/snbl.git
```

### 安裝依賴

```sh
npm install
```

### 啟動開發環境

```sh
npm run serve
```
